
# TodoList App

A simple TodoList app built with **Jetpack Compose** and **Material 3**. It allows users to add, delete, and mark todos as done. The app supports **dark mode**, **reversed todo order**, and **persistent settings** using **DataStore**.

---

## Features

- Material 3 epxressive support
- Android 14 predictive back gesture support
- Reverse the todo list order via settings.
- Switch between **System, Light, and Dark** themes.
- Settings and todos are stored persistently using **DataStore**.
- Modern UI built with **Material 3 Expressive Compose components**.
- Smooth animations for adding/removing todo items.
